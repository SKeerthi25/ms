# MS TVK LTD | Premium Photography Portfolio
A high-end, professional photography platform built with **React**, **Vite**, and **Framer Motion**.

## 🚀 Getting Started

### 1. Installation
Ensure you have [Node.js](https://nodejs.org/) installed, then run:
```bash
npm install
```

### 2. Development Mode
To start the project locally with hot reloading:
```bash
npm run dev
```
The site will be available at `http://localhost:5173`.

### 3. Build for Production
To create a production-ready, optimized bundle in the `dist` folder:
```bash
npm run build
```

### 4. Preview the Build
To see exactly what the production build will look like:
```bash
npm run preview
```

## 📂 Project Structure
- `src/pages/` - All 10+ page components (Home, Gallery, Services, etc.)
- `src/components/` - Reusable UI elements (Header, Footer, GalleryGrid)
- `src/data/` - Centralized data for the 120+ image gallery
- `src/style.css` - Global design system (Dark & Gold theme)
- `public/` - Static assets (images, favicon, sitemap.xml)

## 🎨 Features
- Dark premium theme with gold accents
- Glassmorphism design elements
- Smooth scroll and page animations (Framer Motion)
- Dynamic 120+ image gallery with filtering and lightbox
- SEO-ready with `sitemap.xml` and metadata
